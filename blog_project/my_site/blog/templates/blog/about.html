Creazione Progetto: django-admin startproject [nome progetto]
Creazione di una App: python manage.py startapp [nome_app]
Installazione della app nel portale: aggiungerla alla sezione INSTALLED_APPS di settings.py del progetto Ed effettuare il: python manage.py makemigrations (crea un file in migrations della app, preparando il terreno alla successiva migrazione) Quindi lanciare: python manage.py migrate per renderle effettive scrivendole sul database
Startup server: python manage.py runserver
Creazione Superuser: python manage.py createsuperuser
Per applicare le migrazioni ("collega" le app del progetto al nostro database): python manage.py migrate
